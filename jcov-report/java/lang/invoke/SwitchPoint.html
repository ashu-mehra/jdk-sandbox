 36

  * <p>
 38   
  * A single switch point may be used to control any number of method handles.
 39   
  * (Indirectly, therefore, it can control any number of call sites.)
 40

 41

 43   
  * When a guarded delegator is created from a guarded pair, the pair

 45

 46

 47

 48
